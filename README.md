c5-MobileDetectHelper
=====================

This Helper is completly based on the great PHP class "[PHP Mobile Detect](http://mobiledetect.net/)" by Serban Ghita and Victor Stanciu.

Installation
------------

[Download](https://github.com/axelwehner/c5-MobileDetectHelper/zipball/master) or clone a copy of the c5-MobileDetectHelper, extract it and put the `mobile_detect.php` file in your sites helpers directory.
